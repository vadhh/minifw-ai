# Test Reports

This directory holds QA test reports as defined in [QA_TEST_PLAN.md](../../QA_TEST_PLAN.md).

## File Naming Format

```
YYYY-MM-DD_<test_type>.md
```

Examples:
- `2026-03-15_unit_tests.md`
- `2026-03-16_latency_benchmark.md`
- `2026-03-20_detection_rate.md`

## Requirements

- Each test execution must produce a report file in this directory.
- Reports must follow the template defined in `QA_TEST_PLAN.md`.
- This directory must not be empty when Stage 4 begins.
- Stage 4 sign-off is recorded in `stage4_signoff.md` in this directory.
